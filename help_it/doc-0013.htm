Scarica il file di instalazione adatto al tuo SO da https://sourceforge.net/projects/dc2m/
Nel tab FILES puoi trovare tutte le versioni disponibili.

Linux
DEB package for DEBIAN and derivate is provided. Download it and simply make double click on it to run the package manager of your system.

Also prompt installation is possible, you can use dpgk
sudo dpkg -i dc2m_1_.... .deb

Windows
SETUP... .exe installer is provided. Download it and simply make double click on it and follow the instrucions.
